    分组

    题目描述

    dd 当上了宣传委员，开始组织迎新晚会，已知班里有 n 个同学，每个同学有且仅有一个擅长的声部，把同学们分成恰好 m 组，为了不搞砸节目，每一组里的同学都必须擅长同一个声部，当然，不同组同学擅长同一个声部的情况是可以出现的，毕竟一个声部也可以分成好几个 part 进行表演，但是他不希望出现任何一组的人过多，否则可能会导致场地分配不协调，也就是说，她希望人数最多的小组的人尽可能少，除此之外，对组内人员分配没有其他要求，她希望你告诉她，这个值是多少，如果无法顺利安排，请输出 -1

    输入

    第一行两个数个数 n, m ( 1 ≤ m ≤ n ≤ 100000 )表示人数
    接下来一行 n 个数, a[i] ( 1 ≤ a[i] ≤ n )表示第 i 个学生的擅长声部

    输出

    输出一个数，表示人数最多的小组的人数

    样例

    Input

    5 3
2 2 3 3 3

    Output

    2

    解题思路

    1. 题意很简单，但不太好想去二分
    2. 先统计每一组的人数，并进行特判
    3. 然后以每组人数的最大值作为二分对象，从 n / 2 开始不断二分找最后组数 == m的数

    AC代码

    #include <iostream>
#include <algorithm>

using namespace std;

const int N = 1e5 + 10;

int a[N], g[N], cnt[N];

int n, m, k;

bool check(int x)
{
    int group = 0;
    for (int i = 0; i < k; i ++ )
    {
        group += cnt[i] / x + bool(cnt[i] % x);//上取整
    }

    if (group <= m) return true;//要找组数 == m 的，<=、>=都可，但 l，r 的取值会变
    else return false;
}

int main()
{
    cin >> n >> m;

    for (int i = 0; i < n; i ++ )
    {
        scanf("%d", &a[i]);
        g[a[i]] ++ ;
    }

    for (int i = 0; i < n; i ++ )
    {
        if(g[i]) cnt[k ++ ] = g[i];//用新的数组去存每一组及其人数
    }

    if (k > m)//分的组大于已有的组数，无法分组
    {
        puts("-1");
        return 0;
    }

    int l = 0, r = n - 1;
    while (l < r)
    {
        int mid = l + r >> 1;// mid 为 每组人数的最大值
        if (check(mid)) r = mid;
        else l = mid + 1;
    }

    cout << r << endl;

    return 0;
}

    过桥

    题目描述

    dd 被困在了一个迷幻森林，现在她面前有一条凶险的大河，河中央有 n 个神奇的浮块，浮块按 1 ∼ n 顺序标号，但两两并不相接，第 i 个浮块上有一个数字 a[i] ，可能是正数，也可能是负数,每块浮块都附带一个魔法结界用于传送，当 a[i] 为正数时，dd 可以选择传送到第 i + k ( 1 ≤ k ≤ a[i] )个浮块上,当 dd 抵达 n 号浮块时才可以顺利脱身，显然不管 a[n] 是多少，都没有任何意义，当 a[i] 为负时，dd 只能选择标号小于等于 i + a[i] 的任意一块浮块进行传送，当 i + a[i] < 1 时，默认只能传送到 1 的位置，每次传送都会花费1s的时间，随着时间的流逝，迷雾森林的空气会被逐渐榨干，她现在在1号浮块，她想知道，她最快多久能顺利脱身，如果始终无法逃脱，请输出 −1

    输入

    第一行一个数 n ( 2 ≤ n ≤ 2000 )
    接下来一行n个数a[i] (1≤|a[i]|≤2000)表示浮块上的数字

    输出

    输出一行，表示对应的答案

    样例

    Example 1

    Input

    4
2 2 -1 2

    Output

    2

    Note

    1跳到2,1s
2跳到4,1s
共2s

    Example 2

    Input

    2
-1 -2

    Output

    -1

    解题思路

    1. 一看就知道是DP
    2. 状态表示：dp[i] ：表示在第 i 个点到终点的步数（求最小值）
    3. 状态计算：dp[i] = min(dp[i], dp[j + 1])（ j 为 i 点能到达的地方）[从后往前]
    4. 特判：a[i] 为负数的情况不用考虑

    AC代码

    #include <iostream>
#include <cstring>

using namespace std;

const int N = 2010;

int a[N], dp[N];

int main()
{
    int n;
    cin >> n;

    for (int i = 1; i <= n; i ++ )//除了 -1, 0，0x3f3f3f3f 以外，其它都不能用 memset
        dp[i] = 100000;

    for (int i = 1; i <= n; i ++ )
    {
        scanf("%d", &a[i]);   
        if (i + a[i] >= n) dp[i] = 1; //欲处理出能直接到达终点的转移站 
    }

    for (int i = n - 1; i >= 1; i -- )//从后往前 处理 从该点到后面点的步数
    {
        if (a[i] < 0) continue;//易推，不能走小于零的点
        int cnt = i + a[i];

        for (int j = i; j <= cnt; j ++ )//从后往前的每个中转站 向后逐个更新可到达的点数的步数
            dp[i] = min(dp[i], dp[j] + 1);
    }

    if(dp[1] == 100000) puts("-1");
    else cout << dp[1] << endl;

    return 0;
}
