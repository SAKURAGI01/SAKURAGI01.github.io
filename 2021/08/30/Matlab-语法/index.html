    注释

    1. ctrl + r：注释
    2. ctrl + t：取消注释

    清除

    1. clear：清除工作区

    2. clc：清除命令窗口

    3. 常用：clear;clc

      ; 的作用：

      1. 表示不显示运行结果
      2. 把两个语句分隔开

    输入输出

    1. input：等同于scanf

    2. disp：等同于printf

    字符串合并

    1. strcat(‘str1’, ‘str2’)
    2. [‘str1’, ‘str2’]

    将数字转换成字符串

    num2str

    矩阵中各元素的和与积

    1. sum(A, 1)：按列求和（表示维度，dim = 1）
    2. sum(A, 2)：按行求和（表示维度，dim = 2）
    3. sum(A(:)) ：对整个矩阵求和
    4. prod(A, 1)：按列相乘

    sum(A)表示默认按列求和

    矩阵中的元素

    1. A(1, 2)：第一行第二列的元素
    2. A(1, :) ：第一行中的所有元素
    3. A([2, 5], :)：第二和五行所有元素
    4. A(2:end,:)：第二至最后一行所有元素
    5. A(2:2:5, :)：第二和四行所有元素(等差数列)
    6. A(:)：A的所有元素（输出为列）

    size()函数

    size(A)：返回一个矩阵的大小（行数与列数）

    repmat()函数

    repmat(A, m, n)：把矩阵A叠成m * n块

    矩阵中的运算

    1. A \ B = A * inv(B)：矩阵中的乘除
    2. A .* B, A ./ B：矩阵对应元素的乘除

    求矩阵的特征值和特征向量

    E = eig(A)：求特征值，构成列

    [V, D] = eig(A)：求特征值D及其特征向量V，构成矩阵

    find()函数

    返回矩阵中不为零的位置

    find(A, 2)：寻找前两个不为零的位置

    矩阵在matlab中是以列的形式进行存储

    [r, c] = find(A)：返回两个列向量，以行和列的形式一一对应矩阵中不为零的位置
