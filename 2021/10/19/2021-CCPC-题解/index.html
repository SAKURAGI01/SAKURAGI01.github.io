    Problem F. Nun Heh Heh Aaaaaaaaaaa

    题目描述

    Vasily Tadokorov is a stringologist. He thinks a string is fragrant if it can be divided into two parts — nunhehheh as the prefix and a number of (excluding 0) a as the suffix. For example, nunhehhehaaaaaa is fragrant, but nunhehheh and nunhehhehoooaaa are not fragrant. Today Vasily Tadokorov has some strings consisting of lowercase English letters. For each string, he wants to know how many subsequences of this string are fragrant. A string a is a subsequence of a string b if a can be obtained from b by deletion of several (including 0) characters.

    输入

    The first line contains an integer T (1 ≤ T ≤ 1000), denoting the number of strings. Each of the next T lines contains a string S (1 ≤ |S| ≤ 105 ) consisting of lowercase English letters. The total length of the strings in the input will not exceed 106 .

    输出

    For each of the given T strings, output the answer modulo 998244353.

    样例

    Input

    2
nunhehhehahaahahahahahahaahaahahahahha
nunhehhehhehhahaahahahaahaahahaaaahaa

    Output

    114514
1919810

    解题思路

    1. 这是一道动态规划的题目

    2. 关键是理解它的状态转移的方式，写出状态转移方程

    3. dp[i][j] 代表主串 a 前 i 个字符串可以组成子串 s 前 j 个字符串最多个数

    4. 后缀的 a 的个数为 C(1, n) + C(2, n) + … + C(n, n) = 2 ^ n - 1

    5. 状态方程：

      当 a[i] != s[i], dp[i][j] = dp[i - 1][j]

      当 a[i] == s[i], dp[i][j] = dp[i - 1][j] + dp[i - 1][j - 1]

      当对应字符不相等，传递第 i - 1 状态下的重复子串

      当对应字符相等，在第 i - 1 状态之外，再加上 i - 1 状态下， 出现前 j - 1 字符串的个数

    AC代码

    #include <iostream>
#include <cstring>

using namespace std;

typedef long long LL;

const int N = 1e5 + 10, MOD = 998244353;

char a[N];

LL dp[N][9];

int qmi(int a, int b, int p)
{
    int res = 1;

    for (; b; b >>= 1)
    {
        if (b & 1) res = (LL) res * a % p;
        a = (LL) a * a % p;
    }

    return res;
}

int main()
{
    int n, suma;
    LL res;
    cin >> n;

    while (n--)
    {
        scanf("%s", a + 1);

        int n = strlen(a + 1);

        for (int i = 1; i <= n; i ++ )
        {
            dp[i][1] = (dp[i - 1][1] + (a[i] == 'n')) % MOD;
            dp[i][2] = (dp[i - 1][2] + (a[i] == 'u') * dp[i - 1][1]) % MOD;
            dp[i][3] = (dp[i - 1][3] + (a[i] == 'n') * dp[i - 1][2]) % MOD;
            dp[i][4] = (dp[i - 1][4] + (a[i] == 'h') * dp[i - 1][3]) % MOD;
            dp[i][5] = (dp[i - 1][5] + (a[i] == 'e') * dp[i - 1][4]) % MOD;
            dp[i][6] = (dp[i - 1][6] + (a[i] == 'h') * dp[i - 1][5]) % MOD;
            dp[i][7] = (dp[i - 1][7] + (a[i] == 'h') * dp[i - 1][6]) % MOD;
            dp[i][8] = (dp[i - 1][8] + (a[i] == 'e') * dp[i - 1][7]) % MOD;
            dp[i][9] = (dp[i - 1][9] + (a[i] == 'h') * dp[i - 1][8]) % MOD;
        }

        res = 0, suma = 0;
        for (int i = n; i >= 0; i -- )
        {
            if (a[i] == 'a')//1 + 2 + 4 = 2 ^ 3 - 1，用这种方法递归运算
            {
                suma ++ ;
                res = (res + dp[i][9] * qmi(2, suma - 1, MOD) % MOD) % MOD;
            }
        }
            cout << res << endl;
    }

    return 0;
}

    LeetCode 115. 不同的子序列

    题目描述

    给定一个字符串 s 和一个字符串 t ，计算在 s 的子序列中 t 出现的个数。

    字符串的一个 子序列 是指，通过删除一些（也可以不删除）字符且不干扰剩余字符相对位置所组成的新字符串。（例如，”ACE” 是 “ABCDE” 的一个子序列，而 “AEC” 不是）

    • 0 <= s.length, t.length <= 1000
    • s 和 t 由英文字母组成

    题目数据保证答案符合 32 位带符号整数范围。

    样例1

    Input

    rabbbit
rabbit

    Output

    3

    样例2

    Input

    babgbag
bag

    Output

    5

    解题思路

    1. 这题的思路和上题的基本一致
    2. 注意要初始化，子串为空时，默认所有主串的dp值为1

    AC代码

    #include <iostream>
#include <cstring>

using namespace std;

const int N = 1010;

char s[N], t[N];

int dp[N][N];


int main()
{
    scanf("%s%s", s + 1, t + 1);

    int n = strlen(s + 1);
    int m = strlen(t + 1);

    for (int j = 0; j <= m; j ++ ) dp[0][j] = 0;
    for (int i = 0; i <= n; i ++ ) dp[i][0] = 1;

    for (int i = 1; i <= n; i ++ )
    {
        for (int j = 1; j <= m; j ++ )
        {
            if (s[i] == t[j]) dp[i][j] = dp[i - 1][j] + dp[i - 1][j - 1];
            else dp[i][j] = dp[i - 1][j];
        }
    }

    cout << dp[n][m] << endl;

    return 0;
}
