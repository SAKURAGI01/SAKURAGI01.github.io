    HTML（Hyper Text Markup Langusage）

    超文本标记语言

    HTML是用来描述网页的语言，它不是编程语言，而是一种标记语言——一套标记标签。

    HTML文档包含了HTML标签及文本内容，也称web页面（网页）

    HTML标签是由尖括号包围的关键字，并是成对出现的，前一个叫开始标签（开放标签），后一个叫结束标签（闭合标签）。

    一个小小的例子

    生成后：

    HTML标题

    <h1>我是最大的标题</h1>

    <h6>我是最小的标题</h6>

    标题大小由<h6>到<h1>依次增大

    标题对齐样式

    <h1 align=”center”>我是最大的标题</h1>

    aligen属性应添加至标题标签的后面，此时为居中。

    同样地，left为居左，right为居右。

    如图所示：

    属性

    1.属性是用来设置标签中的内容如何显示

    2.属性与标签名及其它属性用空格隔开

    3.若有属性值应该用引号引起来

    4.属性名与属性值成名值对

    例如 <h1>这是<font color=”green” size=”3”>绿色</font></h1>

    HTML段落

    <p>我是段落</p>

    <p>我是另一个段落</p>

    可以用<p>来实现分段。

    虽然不写</p>时浏览器还能识别出来，但养成好习惯，最好记住吧。

    <p>我是段落<br>我是另一个段落

    用<br>也可以进行换行。

    对文字的处理

    1.<em>我是斜体</em>

    2.<u>我是带下划线的文字</u>

    3.<strike>我是带删除线的文字</strike>

    4.<sup>我是上标</sup>

    5.<sub>我是下标</sub>

    6.<font 属性1 属性2>我是有属性的文字</font>

    font 用于 规定文本的字体、字体尺寸、字体颜色

    对段落的处理

    1.<pre>我是原始排版标签</pre>

    <pre>用于保留选定内容的原始排版，如空格、换行。

    2.空格：&nbsp

    3.水平线：<hr>

    HTML超链接

    <a href="url（我是网址）" target=""(我是打开方式)>我是链接文本</a>

    显示结果： 我是我自己

    在这里，href也是一个属性。

    打开方式

    1._blank:新建一个窗口打开

    2._parent:在上一级窗口中打开

    3._self:在同一窗口中打开（默认值）

    4._top:在浏览器的整个窗口打开

    id属性

    id属性又叫name属性，用来规定锚的名称，来创建HTML页面的书签，点击可直接跳转至锚的位置。

    例如：

    先在HTML文档中插入ID：

    <a id="result">我是结果</a>

    再在HTML文档中创建一个链接：

    <a href="#result">我能跳至结果处</a>

    有几个特殊的：

    <a href="#top">我能跳至开始处</a>
    <a href="#foot">我能跳至结束处</a>

    HTML图像

    <img src="我是图片的地址" alt="我是图片的替换文本">

    1.<img>是空标签，即没有闭合标签

    2.src(sourse):值为图像的URL地址

    3.alt（alternative）：若无法加载图片，为读者备用的文字

    width和height可以定义图像的长和宽。

    图像映射

    图像映射是将图像划分成不同的区域，并赋予其超链接设置。

    例如：

    先定义映射图像名称：

    <img src="我是图像地址" usemap="我是映射图像名称>
