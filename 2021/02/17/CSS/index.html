    CSS概念

    CSS (Cascading Style Sheets)

    层叠样式表

    CSS是用来设置网页中HTML元素的样式。

    网页是多层的结构，CSS可以为每一层设置样式，我们看到的是最外层。

    CSS使用方式

    内联样式表

    1.内联样式也称行内样式

    2.内联样式只对一个标签生效，一般情况下不用

    3.每个样式都名值对应，后面两个样式表同样适用

    <元素名 style="样式名1:样式值1；样式名2:样式值2；..."></元素名>

    举个例子：

    我是绿色

    源代码：

    <p style="color:green; font-size: 20px;">我是绿色</p>

    内部样式表

    1.内部样式将样式编写到<head>中的<style>标签里

    2.内部样式可以为多个标签设置样式，但只对当前页面生效

    3.通过CSS中的选择器选中元素，并对其设置样式

    <!DOCTYPE html><!--声明为 HTML5 文档-->
<html><!-- HTML 页面的根元素-->
<head><!--包含了文档的元（meta）数据-->
<meta charset="utf-8"><!-- 定义网页编码格式为 utf-8-->
<title>我是网页头</title><!-- 定义文章的标题-->
<style>/*<head>中的<style>标签里*/
/*注意注释用法的改变*/
p1/*选择器名1*/
{
color:yellow;/*样式名1：样式值1*/
font-size:40px;/*样式名2：样式值2*/
}
p2/*选择器名2*/
{
 color:green;/*样式名1：样式值1*/
 font-size: 20px;/*样式名2：样式值2*/
}
</style>
</head>

<body><!--包含了页面的可视内容-->

<p1>大黄</p1><!--定义一个黄色的大段落-->
 <br><!--换行-->
<p2>我是小绿</p2><!--定义一个绿色的小段落-->

</body>
</html>

    生成后：

    外部样式表

    1.外部样式将CSS样式编写到外部的CSS文件中（后缀名为css的文件）

    2.外部样式需通过link标签引入样式，link标签位于<head>标签中

    3.外部样式可以为不同的页面设置样式，开发中最佳方式

    4.外部样式有效地使用浏览器缓存机制，提高用户体验

    <link rel="stylesheet" href="样式表的路径">

    注：

    rel：relationship,用于定义链接文件

    stylesheet：样式表，即样式调用

    CSS基础语法

    选择器

    用于选中页面中的元素

    声明块

    1.用于为元素设置样式，用{}包围

    2.由一个或多个声明组成，以；隔开

    3.声明是一个名值对结构，即样式名对应样式值，以：隔开

    常用选择器

    元素选择器

    用标签名选中元素，上例便是

    id选择器

    1.用id属性值选中元素

    2.语法：#id属性值{}

    3.id只能选中一个元素

    例：

    <style>标签中：

    #green{color：green；}

    <body>标签中：

    <p id="green">我是绿色</p>

    类选择器

    1.用class属性值选中元素

    2.语法：.class属性值{}

    3.class可以选中一组元素

    4.一个元素可以拥有多个class，但要用空格隔开

    <style>标签中：

    .green{color:green;}
.big{font-size:40px;}

    <body>标签中：

    <p class="green">我是绿色</p>
<p class="green big">我是大绿</p>

    通配选择器

    1.选中页面中的所有元素

    2.语法：*{}

    复合选择器

    交集选择器

    1.选中同时复合多个条件的元素

    2.语法：选择器1选择器2…{}

    并集选择器

    1.并集选择器又叫选择器分组

    2.同时选中多个选择器对应的元素

    3.语法：选择器1，选择器2，…{}

    关系选择器

    关系

    1.父元素：直接包含子元素的元素

    2.子元素：直接包含父元素的元素

    3.祖先元素：包含后代元素的元素

    4.后代元素：包含祖先元素的元素

    5.兄弟元素：拥有相同的父元素

    如上例便可知：

    1.大div是p，span,小span的祖先元素，是p，span的父元素

    2.小span是大div,p,span的后代元素，是p的子元素

    3.div是span的兄弟元素

    后代选择器

    1.选中指定元素的指定后代元素

    2.语法：祖先 后代

    <style>标签中：

    div span{color:green;}

    如上代码将显示如下:

    继承：

    祖先元素设置的样式会应用到它的后代元素上

    子元素选择器

    1.选中指定父元素的指定子元素

    2.语法：父元素>子元素

    <style>标签中：

    div>span{color:green;}

    如上代码将显示如下:

    兄弟选择器

    1.选择下一个兄弟

    语法：兄+弟（下一个）

    <style>标签中：

    p+span{color:green;}

    此时只有span【弟】变色

    2.选择下边所有兄弟

    语法：兄~弟

    属性选择器

    <style>标签中：

    p[title]{color:green;}
p[title=red]{color:red;}
p[title^=ye]{color:yellow;}
p[title$=ue]{color:blue}
p[title*=ma]{color:tomato;}

    <body>标签中：

    <p title="green">我是绿色</p>
<p title="red">我是红色</p>
<p title="yellow">我是黄色</p>
<p title="blue">我是蓝色</p>
<p title="tomato">我是番茄色</p>

    如上代码将显示如下:

    1.[属性名]：选择含有指定属性的元素

    2.[属性名=属性值]：选择指定属性值的元素

    3.[属性名^=属性值]：选择属性值以指定值开头的元素

    4.[属性名$=属性值]：选择属性值以指定值结尾的元素

    5.[属性名*=属性值]：选择属性值中含有指定值的元素

    Q：一开始已经表明所有p[title]为绿色，为什么不全是绿色？

    A：根据选择器的权重，p[title]的优先级没下面的高，一般的，描述越详细，优先级越高。

    样式冲突：通过不同选择器选中相同元素，设置不同的值，发生了样式冲突

    选择器的权重：

    内联样式：1,0,0,0

    id选择器：0,1,0,0

    (伪)类选择器：0,0,1,0

    元素选择器：0,0,0,1

    通配选择器：0,0,0,0

    继承的样式：没有优先级

    样式后面添加！important，则获取最高优先级，超过内联样式

    伪类选择器

    伪类的定义

    用来描述一个元素的特殊状态

    例如：最后一个子元素、被点击的元素……

    第几个元素

    1.同类型元素中的次第

    第一个：

    :first-of-type

    最后一个：

    :last-of-type

    第n个：

    :nth-of-type(n)

    i.若为偶数

    :nth-of-type(2n或even)

    ii.若为奇数

    :nth-of-type(2n+1或odd)

    2.子元素中的次第

    第n个：

    :nth-child(n)

    其它的，和上述同理

    否定伪类：

    :not()

    a元素中的伪类

    1.没访问过的链接

    :link

    2.访问过的链接（只可修改链接的颜色）

    :visited

    3.鼠标移入的状态

    :hover

    4.鼠标点击的状态

    :active

    <style>标签中：

    a:visited{color:orange;}

    伪元素选择器

    1.第一个字母

    ::first-letter

    2.第一行

    ::first-line

    3.选中的内容

    :: selection

    4.元素的开始

    ::before

    5.元素的结束

    ::after

    <q>引用的内容</q>的用法（前后添加双引号）基本等同于4和5，添加的内容不能被选中

    盒子模型

    CSS将页面中所有的元素设置为一个个矩形的盒子，布局就变成了不同的盒子放在不同的位置，每个盒子都由内容区（content），内边距（padding），边框（border），外边距（magin）组成。

    如图所示

    内容区(content)

    元素中的子元素和文本、图像内容在内容区排列

    1.width：内容区的宽度

    2.height：内容区的高度

    边框(border)

    在盒子的边缘上围绕着盒子，边框的大小影响整个盒子的大小

    1.border-width：边框的宽度

    作用：可以指定四个方向的边框宽度(每个值之间用空格隔开)

    这种方法同样适用于color,style

    默认值：3px

    四个值：上 右 下 左

    三个值：上 左右 下

    两个值：上下 左右

    一个值：上下左右

    还可以用border-xxx-width单独指定一边的宽度

    top：上 right：右 left：左 bottom：下

    2.border-color：边框的颜色

    3.border-style：边框的样式

    i.solid：实线

    ii.dotted：点状虚线

    iii.dashed：虚线

    iiii.double：双线

    默认值：none 没有边框

    border简写属性，也可以单独指定一边

    例如：

    border-top:10px dashed red;

    内边距(padding)

    内容区和边框之间的距离，这同样有四个方向、拥有简写属性，会影响到盒子的大小，会附上背景颜色

    外边距(margin)

    在盒子的边框之外，不会影响盒子的大小，但会影响盒子的位置，实际占用的空间，同样拥有四个方向、拥有简写属性。

    1.margin-top:上外边距，设置正值，元素会向下移

    2.margin-right:默认不会产生效果

    3.margin-bottom:下外边距，设置正值，下边元素向下移

    4.margin-left:左外边距，设置正值，元素会向右移

    水平布局

    元素在其父元素，必须满足以下条件：

    margin-left+border+padding-left+width+padding-right+border+margin-right=父元素宽度
