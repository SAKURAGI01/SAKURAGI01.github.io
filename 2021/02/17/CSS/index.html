    CSS概念

    CSS (Cascading Style Sheets)

    层叠样式表

    CSS是用来设置网页中HTML元素的样式。

    网页是多层的结构，CSS可以为每一层设置样式，我们看到的是最外层。

    CSS使用方式

    内联样式表

    1.内联样式也称行内样式

    2.内联样式只对一个标签生效，一般情况下不用

    3.每个样式都名值对应，后面两个样式表同样适用

    <元素名 style="样式名1:样式值1；样式名2:样式值2；..."></元素名>

    举个例子：

    我是绿色

    源代码：

    <p style="color:green; font-size: 20px;">我是绿色</p>

    内部样式表

    1.内部样式将样式编写到<head>中的<style>标签里

    2.内部样式可以为多个标签设置样式，但只对当前页面生效

    3.通过CSS中的选择器选中元素，并对其设置样式

    <!DOCTYPE html><!--声明为 HTML5 文档-->
<html><!-- HTML 页面的根元素-->
<head><!--包含了文档的元（meta）数据-->
<meta charset="utf-8"><!-- 定义网页编码格式为 utf-8-->
<title>我是网页头</title><!-- 定义文章的标题-->
<style>/*<head>中的<style>标签里*/
/*注意注释用法的改变*/
p1/*选择器名1*/
{
color:yellow;/*样式名1：样式值1*/
font-size:40px;/*样式名2：样式值2*/
}
p2/*选择器名2*/
{
 color:green;/*样式名1：样式值1*/
 font-size: 20px;/*样式名2：样式值2*/
}
</style>
</head>

<body><!--包含了页面的可视内容-->

<p1>大黄</p1><!--定义一个黄色的大段落-->
 <br><!--换行-->
<p2>我是小绿</p2><!--定义一个绿色的小段落-->

</body>
</html>

    外部样式表

    1.外部样式将CSS样式编写到外部的CSS文件中（后缀名为css的文件）

    2.外部样式需通过link标签引入样式，link标签位于<head>标签中

    3.外部样式可以为不同的页面设置样式，开发中最佳方式

    4.外部样式有效地使用浏览器缓存机制，提高用户体验

    <link rel="stylesheet" href="样式表的路径">

    注：

    rel：relationship,用于定义链接文件

    stylesheet：样式表，即样式调用

    CSS基础语法

    选择器

    用于选中页面中的元素

    声明块

    1.用于为元素设置样式，用{}包围

    2.由一个或多个声明组成，以；隔开

    3.声明是一个名值对结构，即样式名对应样式值，以：隔开

    常用选择器

    元素选择器

    用标签名选中元素，上例便是

    id选择器

    1.用id属性值选中元素

    2.语法：#id属性值{}

    3.id只能选中一个元素

    例：

    <style>标签中：

    #green{color：green；}

    <body>标签中：

    <p id="green">我是绿色</p>

    类选择器

    1.用class属性值选中元素

    2.语法：.class属性值{}

    3.class可以选中一组元素

    4.一个元素可以拥有多个class，但要用空格隔开

    <style>标签中：

    .green{color:green;}
.big{font-size:40px;}

    <body>标签中：

    <p class="green">我是绿色</p>
<p class="green big">我是大绿</p>

    通配选择器

    1.选中页面中的所有元素

    2.语法：*{}

    复合选择器

    交集选择器

    1.选中同时复合多个条件的元素

    2.语法：选择器1选择器2…{}

    并集选择器

    1.并集选择器又叫选择器分组

    2.同时选中多个选择器对应的元素

    3.语法：选择器1，选择器2，…{}

    关系选择器

     <div>
我是大div【祖】
         <p>
         我是p【兄】
                <span>
                我是小span【孙】
                </span>
         </p>

         我是span【弟】

    关系

    1.父元素：直接包含子元素的元素

    2.子元素：直接包含父元素的元素

    3.祖先元素：包含后代元素的元素

    4.后代元素：包含祖先元素的元素

    5.兄弟元素：拥有相同的父元素

    如上例便可知：

    1.大div是p，span,小span的祖先元素，是p，span的父元素

    2.小span是大div,p,span的后代元素，是p的子元素

    3.div是span的兄弟元素

    后代选择器

    1.选中指定父元素的指定子元素

    2.语法：祖先 后代
