    聚焦爬虫

    爬取页面中的指定页面内容

    ​ 编码流程

    1. 指定url
    2. 发起请求
    3. 获取响应数据
    4. 数据解析
    5. 持久化存储

    数据解析分类

    1. 正则

    2. bs4

    3. xpath

    数据解析原理概述

    解析的文本在标签之间或者标签对应的属性中进行存储

    1. 进行指定标签的定位
    2. 标签或者标签对应的属性中存储的数据进行提取（解析）

    bs4进行数据解析

    数据解析的原理

    1. 标签定位
    2. 提取标签、标签属性中存储的数据值

    bs4数据解析的原理

    1. 实例化一个BeautifulSoup对象，并且将页面源码数据加载到该对象中
    2. 通过调用BeautSoup对象中的相关的属性或者方法进行标签定位和数据提取

    环境安装

    pip install bs4

    pip install lxml

    如何实例化BeautifulSoup对象

    1. 将本地的html文档中的数据加载到该对象中
    fp = open('./sogou.html', 'r', encoding='utf-8')
soup = BeautifulSoup(fp,'lxml')
    1. 将互联网上获取的页面源码加载到该对象中
    page_text = requests.get(url = url).text
soup = BeautifulSoup(page_text,'lxml')

    提供用于数据加载和解析的属性

    1. soup.tagname

    返回文档中第一次出现的tagname标签

    1. soup.find('tagname')

    返回文档中第一次出现的tagname标签

    1. soup.find('div', class_ ='classname' )

    返回文档中class指定类中的所有属性

    1. soup.find_all('tagname')

    返回符合要求的所有标签（列表）

    1. soup.select('tagname')

    返回拥有含tagname标签的列表

    层级选择器

    soup.select('.classname > tagname1 tagname2')

    空格表示多个层级，大于 表示一个层级

    获取标签中的文本数据 及 属性值

    1. 文本数据

    soup.select('tagname')[0].text

    1. 属性值

    soup.select('tagname')[0]['属性值']

    案例实战

    from bs4 import BeautifulSoup
import requests

if __name__ == "__main__":

    # 指定 url
    url = 'https://www.shicimingju.com/book/sanguoyanyi.html'

    headers = {
        'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/103.0.0.0 Safari/537.36'
    }
    # 发起请求
    page_text = requests.get(url = url, headers = headers)
    page_text.encoding = 'utf-8'
    page_text = page_text.text


    soup = BeautifulSoup(page_text,'lxml')

    li_list = soup.select('.book-mulu > ul > li')

    fp = open("./threecountry.txt", 'w', encoding='utf-8')
    for li in li_list:
        # print(li.text)
        title = li.a.string
        detail_url = "https://www.shicimingju.com" + li.a['href']
        detail_page_text = requests.get(url = detail_url, headers = headers)
        detail_page_text.encoding = 'utf-8'
        detail_page_text = detail_page_text.text
        detail_soup = BeautifulSoup(detail_page_text, 'lxml')
        div_tag = detail_soup.find('div', class_ = 'chapter_content')
        content = div_tag.text
        fp.write(title + ':' + content + '\n')
        print(title, '爬取成功')

    xpath

    xpath解析原理

    1. 实例化一个etree对象，且需要将被解析的页面源码数据加载到该对象中。
    2. 调用etree对象中的xpath方法结合着xpath表达式实现标签的定位和内容的捕获

    环境的安装

    pip install lxml

    xpath表达式

    58二手房

    import requests
from lxml import etree

if __name__ == "__main__":
    url = 'https://nj.58.com/ershoufang/'

    headers = {
        "User-Agent" : "Mozilla/5.0 (Linux; Android 6.0; Nexus 5 Build/MRA58N) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/103.0.5060.66 Mobile Safari/537.36 Edg/103.0.1264.44"
    }

    page_text = requests.get(url = url, headers = headers).text

    tree = etree.HTML(page_text)

    li_list = tree.xpath('//ul[@class="list"]/li')

    fp = open('58.txt', 'w', encoding = 'utf-8')
    for li in li_list:
        #局部解析
        # li.xpath('li/a/div[2]/div[1]/span/text()')
        title = li.xpath('./a/div[2]/div[1]/span/text()')[0]
        fp.write(title + '\n')
